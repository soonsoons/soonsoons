## Projet principal : BaliseTonSite.fr

Je suis le créateur et développeur de [BaliseTonSite.fr](https://balisetonsite.fr), une plateforme d'apprentissage en ligne interactive (LMS) conçue pour accompagner la reconversion professionnelle vers le développement web moderne et l'intégration pragmatique des outils d'IA.

- **Architecture moderne** : Frontend Next.js/React déployé sur Cloudflare Pages, couplé à un backend WordPress headless (PHP).
- **Philosophie pédagogique** : Une formation accessible, axée sur la pratique et les compétences réelles du marché.
- **Modèle équitable** : Achat unique avec accès à vie, sans abonnement.

[Découvrir la plateforme](https://balisetonsite.fr)
